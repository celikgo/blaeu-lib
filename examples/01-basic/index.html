BlaeuMap · example 01

Basic usage

A bare kernel plus five plugins — snap, draw, select, history, ui. No preset, so nothing is hidden.

Try this

  1. Pick Polygon in the toolbar (top-left). Click to place vertices, double-click to finish.
  2. Draw a parcel that touches an existing corner. Watch the cursor jump onto it.
  3. Hold Alt while you click: snapping is suppressed for that one event, as in every CAD package on earth.
  4. Ctrl/⌘+Z to undo, Shift+that to redo.
  5. Pick Select and click a parcel. Shift-click to add to the selection.

Notice

  • The draw plugin cannot snap. It has never heard of the snap plugin and does not import it. Snapping is interaction middleware: it rewrites the pointer position before any tool sees the event. Delete snapPlugin() from src/main.ts and drawing still works — it just stops snapping.
  • Undo is not a draw feature. The history plugin subscribes to the command bus. It knows nothing about polygons.
  • The toolbar was not configured. The UI plugin derives its buttons from whatever tools happen to be registered.
